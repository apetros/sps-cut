## Scripts

### Batch convert PDF to images in Ubuntu

First, install poppler-utils:

    sudo apt-get install poppler-utils

Then, move to the folder with the PDFs and execute:

    for i in *.pdf; do pdftoppm -jpeg -r 300 "$i" "$i".jpg; done
